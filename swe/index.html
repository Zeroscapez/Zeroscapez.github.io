SingWithFriends

↗

A networked, multi-user music control system built in C# and .NET 10. Features an embedded ASP.NET Core Web API (Kestrel) running inside a WinForms desktop app — friends join via room code from any browser on the network, upload songs from their phone, and control playback in real time. Includes custom API key and guest token authentication middleware, request logging middleware, file upload validation, Swagger docs, and a full GitHub Actions CI/CD pipeline with semantic versioned releases.

C# .NET 10 ASP.NET Core Kestrel WinForms NAudio Swagger GitHub Actions CI/CD
Host/guest role system · Custom auth middleware · Logging · v0.1.6
